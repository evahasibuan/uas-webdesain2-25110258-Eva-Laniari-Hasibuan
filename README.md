**Deskripsi:**
Website ini merupakan Company Profile untuk UMKM kuliner **Kebab Dara**, sebuah kedai kebab dan sawarma yang berlokasi di kawasan Panam, Pekanbaru. Website dibuat sebagai tugas UAS mata kuliah Web Desain 2, dengan tujuan mempromosikan menu, harga, dan informasi kontak UMKM secara digital.

## 🍔 Halaman
- **Home** — Hero section dengan tagline, foto produk unggulan, dan tombol ajakan (CTA) untuk menjelajahi menu.
- **Menu Eksklusif** — Daftar menu kebab & sawarma lengkap dengan foto, deskripsi, dan harga.
- **Cerita Kami** — Galeri visual proses produksi serta cerita singkat berdirinya Kebab Dara.
- **Hubungi & Order** — Informasi kontak (Instagram, WhatsApp, Email), alamat kedai dengan embed Google Maps, serta form checkout pemesanan.

## ⚙️ Fitur JavaScript
- Formulir checkout interaktif: pemilihan menu, perhitungan total estimasi harga otomatis, dan tombol kirim pesanan ke WhatsApp.
- Navbar responsif (Bootstrap Navbar Collapse) untuk tampilan mobile.

## 🛠️ Teknologi
- HTML5
- CSS3 + Bootstrap 4/5
- JavaScript
